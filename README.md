# Project myKanban - RESTfull API

REST API for my project managemment website 

### Core

- SpringBoot
    - data JPA
    - security
    - OAuth2
- MySQL

### Requirements

- JDK 8
- MySQL 5.6

### Usage

- clone project: `git clone https://github.com/tinnguyen263/ws_myKanban.git`
- open project with IntelliJ IDEA
- run SpringBoot Application
